# Results manifest

All paths under `results/`. `previous_results/`, `baselines/`, and the reference
`figures/`+`metrics/` entries are **symlinks** into the original audit outputs
(read-only). `reproduced_results/`, and the `smoke*/full100*` metric/figure files
are produced by this project and are regenerable.

## Reference (symlinked) — the numbers we reproduce against
| path | what |
|---|---|
| `results/previous_results/ours_rerun_best/group100_<task>_best/` | OUR method, rerun `*best`, 100 imgs/task, DAPS-aligned. `meta.json` has per-image + mean PSNR/SSIM/LPIPS. **This is the headline "ours".** |
| `results/previous_results/ours_demorunner/<task>_LPIPS_king*/` | OUR method, `demo_runner` pipeline (bit-exact reproduction), 100 imgs/task. |
| `results/previous_results/FINAL_REPORT.md` | full audit + 3-way analysis. |
| `results/previous_results/groups_used/group_100/` | DAPS's exact 100 images (GTs + masks_box + masks_random), `sample_seed=42`, cca crop. |
| `results/baselines/{daps_pixel,ddnm,ddrm,dps,diffpir,fps_smc,psld,resample}/` | baseline recons (proper-crop, DAPS-aligned where applicable). |
| `results/figures/final_3way_grids/` | per-task GT&#124;demo_runner&#124;rerun&#124;DAPS grids + error maps. |
| `results/figures/metric_visualization_grids/` | 15-demo per-task comparison grids. |
| `results/metrics/previous_baseline_metric_comparison.csv` | canonical 15-demo baseline metrics (each-vs-own-GT). |

## Reproduced (generated by this project)
| path | produced by |
|---|---|
| `results/reproduced_results/<mode>/group100_<task>_best/{recons,gts,grid.png,meta.json}` | `run_ours.py` / `reproduce_previous_results.py` |
| `results/metrics/<mode>_<task>.csv` (+ `_summary.json`) | `evaluate.py` |
| `results/metrics/reproduction_<mode>.json` | `reproduce_previous_results.py` (repro-vs-recorded comparison) |
| `results/metrics/existing_<tag>.csv` | `evaluate_existing_results.sh` (re-eval of the reference recons) |
| `results/metrics/baseline_comparison.{md,csv}` | `compare_baselines.py` |
| `results/figures/{smoke,existing_eval}/` | `visualize.py` |

`<mode>` ∈ {`smoke<N>`, `full100`}. `<task>` ∈ {super_resolution, deblur,
motion_deblur, inpainting, inpainting_random}.

Headline per-task reference means live in each run's `meta.json`
(`results/previous_results/ours_rerun_best/group100_<task>_best/`) — numbers are
kept with the results on the reproduction machine, not in this repo.
